Kinds of Links

There are three kinds of links:

  • Absolute Links, like Mozilla Developer Network are a very good resource for web developers.
  • Relative Links, like our extras page
  • Root-Relative Links, like the home page of our website, but those aren't useful to us right now because these are relative to the “root” of the entire website. Not interesting here, because we only work with local html files instead of a web server.
